Mi mostri un esempio di codice per leggere tutte le risposte ad una mail con python?

Ecco un esempio di codice che utilizza la libreria "imaplib" di Python per leggere tutte le risposte ad una specifica email:

import imaplib

# Connessione al server IMAP
imap_server = imaplib.IMAP4_SSL("imap.example.com")
imap_server.login("username", "password")

# Selezione della cartella "Inbox"
imap_server.select("Inbox")

# Ricerca delle email con argomento "example subject"
status, email_ids = imap_server.search(None, 'SUBJECT "example subject"')
email_ids = email_ids[0].split()

# Scaricamento delle risposte per ogni email trovata
for email_id in email_ids:
    status, email_data = imap_server.fetch(email_id, "(RFC822)")
    email_body = email_data[0][1]
    # Elaborazione della risposta
    print(email_body)

# Chiusura della connessione
imap_server.close()
imap_server.logout()

Nota che questo codice assume che il server IMAP utilizzi una connessione SSL sicura e che l'utente fornisca le credenziali corrette per l'accesso. Inoltre per avere tutte le risposte alla mail occorre fare una ricerca per tutte le email con subject "example subject" e poi fare un fetch su tutte le email trovate per recuperare la loro risposta.
